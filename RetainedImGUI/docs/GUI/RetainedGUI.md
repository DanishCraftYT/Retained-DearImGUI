# class RetainedGUI
handles ImGUI boilerplate code.<br>

## public RetainedGUI(GLFWwindow* window, int OpenGLVersion)
constructor for the RetainedGUI class.<br>

### params:
* GLFWwindow* window - the GLFW Window the GUI should be rendered to.<br>
* int OpenGLVersion - the OpenGL version to use.<br>

## public void ImGUINewFrame()
creates a new frame for ImGUI.<br>

## public void ImGUIRender()
renders ImGUI.<br>

## public void ImGUITerminate()
terminates ImGUI.<br>
