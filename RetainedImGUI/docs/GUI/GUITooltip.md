# class GUITooltip
handles Tooltips for GUI Elements.<br>

## public GUITooltip(std::string name, std::string text)
constructor for the GUITooltip class.<br>

### params:
* std::string name - the name of the Tooltip.<br>
* std::string text - the text the Tooltip should display.<br>

## public GUITooltip()
constructor for the GUITooltip class.<br>

## public void render()
renders the Tooltip.<br>

## public GUIElementVisibility visible
determines the Tooltip's visibility state.<br>

## private std::string name
contains the Tooltip's name.<br>

## private std::string text
contains the text the Tooltip should display.<br>

## private std::string uuid
contains the Tooltip's UUID.<br>
