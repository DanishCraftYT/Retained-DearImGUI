# class Separator : public GUIElement
handles separating GUI Elements into segments.<br>

## public Separator(std::string name, std::string text="")
constructor for the Separator class.<br>

### params:
* std::string name - the name of the Separator.<br>
* std::string text="" - the text the Separator should display.<br>

## public Separator()
constructor for the Separator class.<br>

## public void render() override
renders the Separator.<br>

## public std::string text
contains the text the Separator should display.<br>
